XISL Software Developing Kit
--------------------
--------------------

The XISL demonstration application is a simple console application running
under Windows '2000 as well as Windows XP/VISTA. It demonstrates how to acquire data
continuously, how to acquire a sequence, how to create correction files and
how to acquire corrected images.

Files:

XISL.DLL


main.c

------
The demonstration program was build with MS Visual C++ 6.0. If you want to
use your own development environment insert xisl.lib in your project.
The header file declaring the required function prototypes is named "Acq.h".
The main application program source is called "main.c".
